GITHUB

Github is a website where Git projects are stored online.

Git is the tool. Github is the cloud storage for git projects.

Github allows you:

  • Backup your code
  • Share projects
  • Build projects
  • Collaborate

Push Project To Github

Step 1: Create a Repository

  1. Go to github
  2. Click New Repository
  3. Enter a repository name (e.g my-first-project)
  4. Click Create Repository
  5. Leave the page open

Step 2: Open your project in VS Code

Open the folder that contains your project files

Step 3: Open Terminal in VS Code (ctrl + j)

Step 4: Initialize Git

git init

This tells git to start tracking this project

Step 5: Check Files

git status

You will see untracked files

Step 6: Stage files

git add .

add: means move files to the staging area

. : means everything

the staging area is like a waiting room

Step 7: Commit Files

git commit -m "first commit"

commit: means save a snapshot

-m: means message

Example: git commit -m "just created a footer"

Step 8: Connect Local Project to Github

Copy the URL from Github. Example: https://github.com/yourname/my-first-project.git

Run this in your terminal:

git remote add origin https://github.com/yourname/my-first-project.git

remote: means online repository

origin: means nickname for github

Check: git remote -v

Step 9: Rename Branch to Main

git branch -M main

branch: means line of development

main: means primary branch

Step 10: Push to Github

git push -u origin main

push: upload

origin: github repository

main: branch main

Go and refresh your github, these files will now appear on Github.

What do i do every time i make changes?

Suppose you edit your files. Or each time you edit your file.

  1. Check changes: git status
  2. Stage changes: git add .
  3. Commit changes: git commit -m "Updated homepage"
  4. Push changes: git push

That's it. Repeat these four cycle whenever you update a project.
